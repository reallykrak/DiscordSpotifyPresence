🎵 How to get Spotify credentials
  1. Go to developer.spotify.com/dashboard and log in
  2. Click Create App — fill in any name and description
  3. Copy your Client ID and click View client secret
  4. Paste both above and click Save Settings
  5. Now the search bar on Songs tab will work!
Add Songs via Spotify URL
Paste one or more Spotify track links (one per line). You can find these by right-clicking a song in Spotify → Share → Copy Song Link.
